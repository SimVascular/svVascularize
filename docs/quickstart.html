Quickstart tutorial

This short guide shows how to grow a small vascular tree and export 0‑D simulation files.

Generate and visualise a tree

import pyvista as pv
from svv.domain.domain import Domain
from svv.tree.tree import Tree

cube = Domain(pv.Cube())
cube.create()
cube.solve()
cube.build()

t = Tree()
t.set_domain(cube)
t.set_root()
t.n_add(50)

t.show(plot_domain=True)

Export 0‑D simulation files

from svv.simulation.fluid.rom.zero_d.zerod_tree import export_0d_simulation

export_0d_simulation(t, outdir="my_0d")

This writes solver_0d.in and geom.csv in my_0d/. Run the solver with svZeroDSolver.
